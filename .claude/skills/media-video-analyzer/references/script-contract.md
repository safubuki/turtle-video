# analyze_video.py 契約

## モード

| モード | 定型出力 | AIが判断すること |
|---|---|---|
| `summary` | 動画メタデータ、サンプル輝度、隣接サンプル差分の統計 | 追加解析の要否、注目区間 |
| `black-segments` | 平均輝度が閾値以下の連続サンプル区間 | 意図した暗転か不具合か |
| `freeze-segments` | 隣接サンプル差分が閾値以下の連続区間 | 静止画面かフリーズか |
| `transcribe` | 開始・終了時刻付き文字列 | 発話の意味、症状との関連 |

## 計測上の制約

- フレームを `--sample-interval` 秒ごとにサンプリングするため、短い異常を見逃す場合がある。
- `scope=tail` は動画末尾 `--tail-seconds` 秒だけを対象にする。
- 黒判定はRGB平均輝度、フリーズ判定は連続サンプル間の平均絶対差を使う。
- 区間は時刻順で出力し、同じ入力・引数では安定したJSON構造にする。

## 安全性

入力動画は読み取り専用。標準出力が既定で、`--output` は `--force` なしに既存ファイルを上書きしない。依存導入、モデル取得、外部送信はスクリプトから明示実行しない。
